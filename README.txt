# Бэклог команды №4262 (голос и речь)

## Юзер-сториз:
1)	Геннадий, средний возраст, работает в офисе, ему поручили произнести речь на корпоративе.
2)	Василий Петрович, средний возраст, читает лекции в ВУЗе ему поручили временно заменить одного лектора.
3)	Дмитрий, юный предприниматель, хочет найти инвестора для стартапа, ему нужна речь-презентация.
4)	Данил, выпускник готовится к поступлению на работу, осталось пройти собеседование.
5)	Петя, школьник, готовится защитить свой реферат.

## Концептуальный сценарий (версия 2)
Человек уже имеет готовый текст некоторой речи, 
которую хочет отрепетировать, запомнить, но не в слово в слово.
Он скачивает приложение, загружает в него текст и по-памяти воспроизводит его под запись.
После записи речи, она конвертируется в видимый пользователю текст. 
Пользователь также видит информацию о словах-паразитах, длинных паузах, 
среднем темпе речи, итоговой продолжительности, степень близости к исходному тексту по смыслу
и ознакомительные рекомендации по улучшению речи.
